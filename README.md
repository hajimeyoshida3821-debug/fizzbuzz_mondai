# fizzbuzz_mondai

1から20までの数を出力し、3の倍数で「Fizz」、5の倍数で「Buzz」、3と5の公倍数で「FizzBuzz」を出力するJavaScriptスクリプトです。

## 制作・登録プロセス

### 1. 設計
- 以前、オンライン学習プラットフォーム「Progate」にて類似の条件分岐を学習していたため、その記憶をベースに設計しました。

### 2. コーディング
- VS Codeを用いて制作しました。

### 3. テスト
- 記述した `fizzbuzz.js` をterminalで実行し、出力結果が仕様通りになっているかを確認しました。

### 4. 登録について
- GitHub上に新しくパブリックリポジトリを作成し、ブラウザ経由で対象ファイルをアップロードして登録・公開しました。
